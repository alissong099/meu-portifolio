# Portfólio Pessoal - Alisson Gabriel

Este é o meu portfólio pessoal de projetos, desenvolvido como parte dos critérios de avaliação da disciplina de Desenvolvimento Web I, ministrada pelo Professor Ely no IFPI. 

O design do projeto foi construído com uma abordagem minimalista e limpa, inspirado na identidade visual do designer Matt Farley, utilizando uma paleta de cores suave e foco na tipografia.

## 🚀 Tecnologias Utilizadas

*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização avançada, layouts flexíveis (Flexbox) e interações dinâmicas (efeitos Hover).

## 🛠️ Funcionalidades e Critérios Atendidos

*   **Semântica HTML:** Uso correto de tags como `header`, `nav`, `main`, `section`, `article` e `footer`.
*   **Identidade Visual Personalizada:** Aplicação de paleta de cores minimalista baseada em códigos hexadecimais (sem uso de `:root`).
*   **Componentes Obrigatórios:** Inclusão de um formulário de contato estruturado e uma tabela técnica para detalhes dos projetos.
*   **Acessibilidade:** Tags de imagens configuradas com atributos `alt` descritivos e amigáveis para leitores de tela.
